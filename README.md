# Instagram_Data_Analysis
Data Science project analyzing Instagram engagement, hashtags, user activity, and content performance using Python, Pandas, Matplotlib, and Seaborn.
# Instagram Engagement Analysis

## Project Overview

This project analyzes Instagram user activity, post engagement, hashtags, filters, and content performance using Python. The objective is to identify engagement patterns and provide actionable recommendations to improve social media performance.

This project was completed as part of the **InternSpark Data Science Internship**.

---

## Objectives

* Analyze Instagram engagement using likes and comments.
* Identify top-performing posts.
* Analyze hashtag usage and popularity.
* Study user activity patterns.
* Evaluate the impact of Instagram filters.
* Generate recommendations to improve engagement.

---

## Dataset Information

The dataset contains Instagram-related information across multiple tables:

* Users
* Photos
* Likes
* Comments
* Tags
* Photo Tags
* Follows

### Dataset Size

| Dataset    | Records |
| ---------- | ------- |
| Users      | 100     |
| Photos     | 257     |
| Likes      | 8,782   |
| Comments   | 7,488   |
| Tags       | 21      |
| Photo Tags | 501     |
| Follows    | 7,623   |

## Key Findings

* High engagement posts receive significantly more likes and comments.
* Hashtags help improve content visibility.
* Instagram filters influence engagement levels.
* Active users contribute a large portion of content.
* Consistent posting can improve audience interaction.

---

## Recommendations

1. Use relevant and trending hashtags.
2. Maintain a consistent posting schedule.
3. Create engaging captions to encourage comments.
4. Use effective Instagram filters.
5. Monitor top-performing content and replicate successful strategies.

---


## Conclusion

This project demonstrates how data analysis can be used to understand Instagram engagement patterns and support data-driven social media strategies. The insights generated can help improve audience engagement and content performance.
